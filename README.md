# 🕌 الموقع الإبداعي للقرآن الكريم

موقع تفاعلي لقراءة وتدبر آيات القرآن الكريم، مبني باستخدام Streamlit و Quran Cloud API.

## ✨ المميزات

- **📖 قراءة القرآن** - تصفح جميع سور القرآن مع الترجمة الإنجليزية
- **🎯 آية عشوائية** - احصل على آية عشوائية للتدبر اليومي
- **🔍 بحث** - ابحث عن كلمات في آيات القرآن
- **🌙 واجهة عربية** - تصميم أنيق باللغة العربية
